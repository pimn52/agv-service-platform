# 车型规格手册

> 全项目车型参数的唯一来源。所有车型均为 2025-2026 年行业真实量产车型对标，参数取自厂商公开数据。

---

## 设计原则

- **精不在多**：每条业务线 3-5 款，覆盖能力边界即可
- **全真实参数**：不取整、不估算，全部采用厂商公开规格
- **车型决定套餐**（巡游/安防）：车型与兼容套餐强绑定，客户选车即选套餐
- **特殊要求限车型**（物流）：冷链/大件等特殊要求自动过滤不支持该要求的车型

### 车型定位

| 车型 | 适用业务 | 场景定位 |
|------|:--:|------|
| Z2 | 整车 / 散件 | **窄路末端**：社区、校园、老旧小区、非机动车道，最小最灵活 |
| X3 | 整车 / 散件 | **城配主力**：城市末端配送，快递接驳，社区团购 |
| Z5 | 整车为主 | **通用中型**：城市主干道高效配送，可选冷链，应用最广的车型 |
| E6 | 整车专用 | **轻抛大容积·一趟多站**：无立柱设计最大化空间利用率，适合一车跑多站的大容积配送 |
| X6 | 整车为主 | **大容量重载**：商超补货、网点接驳、干线支线转运 |

### 业务约束联动

| 特殊要求 | 可选车型 | 不可选车型 |
|----------|----------|-----------|
| `cold_chain`（冷链） | Z5（选配冷藏）、X6（标配1+选配1） | Z2、X3、E6 |
| `oversized`（超大件） | X6 | Z2、X3、Z5、E6 |
| `fragile`（易碎品） | 全部 | —（所有车型均支持） |
| 散件直送模式 | Z2、X3、Z5、X6（有格口的车型） | E6（无独立格口，不参与散件拼车） |
