# 城市无人车服务平台

城市无人车物流与租车服务平台的客户端 APP（网页版 MVP），模拟手机 APP 使用效果。支持三大核心服务：物流配送、巡游贩卖、安防巡检。

## 快速开始

### 启动开发服务器

```bash
coze dev
```

启动后，在浏览器中打开 [http://localhost:5000](http://localhost:5000) 查看应用。

### 构建生产版本

```bash
coze build
```

### 启动生产服务器

```bash
coze start
```

- **底部4Tab**：首页 / 服务跟踪 / 订单 / 我的
- **子页面栈**：模拟 APP push/pop 导航，通过 `useAppStore` 的 `pageStack` 管理
- **桌面端**：居中显示 375×812 手机模拟框 + 状态栏；移动端全屏

## 订单工作流

### 物流配送

选择配送下单 → 整车/零担（整车选车型） → 填写发货/收货信息 → 计算运费 → 确认支付 → 系统自动调度+规划路线 → 在途追踪+人车交互（装货/卸货/收件/取件） → 订单完成

### 巡游贩卖

选择巡游租车→贩卖 → 选车型+配套套餐 → 填写信息 → 可选路线规划（任务停留点+经过点+路段任务） → 计算费用 → 确认支付 → 调度车辆 → 在途追踪（开始/结束行程、贩卖/暂停、货物操作、路线规划） → 订单完成

### 安防巡检

选择巡游租车→安防 → 选车型+配套套餐 → 填写信息 → 可选路线规划（任务停留点+经过点+路段任务） → 计算费用 → 确认支付 → 调度车辆 → 在途追踪（开始/结束行程、物资操作、设备测试、路线规划） → 订单完成 → 生成巡检报告

## 路线规划功能

路线规划支持三种点位类型：
- **任务停留点**：定时停留 + 执行任务列表
- **非任务停留点**：短暂停留，无任务
- **经过点**：不停留，仅经过

路段任务：在两个点位之间的行驶途中执行任务。

支持拖拽排序、删除点位、预览路线概览、保存草稿。下单前可规划，下单后可修改。
